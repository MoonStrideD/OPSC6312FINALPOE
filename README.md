# BreezyNest - Property Rental Management App
## App Overview
BreezyNest is a comprehensive property rental management application that connects landlords and tenants, making property management seamless and efficient. The app provides features for property listings, messaging, notifications, and rental management.

##  Key Features
### User Roles
Landlords: Manage properties, send notifications, communicate with tenants

Tenants: Browse properties, receive notifications, message landlords

### Property Management
Property Listings: View available properties with detailed information

Search & Filter: Find properties by location, type, price, and amenities

Property Details: Comprehensive property information with images and descriptions

Favorites: Save preferred properties for quick access

### Communication System
Real-time Messaging: Direct communication between landlords and tenants

Broadcast Notifications: Landlords can send announcements to all tenants

Message History: Complete conversation history with timestamps

Read Receipts: Track message status

### Notification System
Rent Reminders: Automated rent due notifications

New Property Alerts: Notifications for new property listings

Maintenance Updates: Important property maintenance information

Custom Messages: Landlords can send custom notifications

### Dashboard Features
Landlord Dashboard: Property management tools and analytics

Tenant Dashboard: Personal rental information and messages

Quick Stats: Property and application statistics
